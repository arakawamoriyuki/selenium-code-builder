Step

ステップとは、実行する作業の単位です。

NavigationやElementなどのメニューからステップを追加することができます

実行するステップの各パラメータを編集し、複数のステップを組み合わせて、テストコードを組み立てていきます

一部を除き、各パラメータは#{変数名}で変数を展開できます

Deleteボタンで削除、ステップ名をドラッグすることで位置を変更できます

Native->Ifなどの青い色のステップはドラッグで子を作ることができます

現在選択中のステップは赤い色で表示されます

Shiftを押下中に選択することで複数のステップを選択できます

Ctrl+Cで選択中のステップをコピー、Ctrl+Vで最後に選択したステップの下にペーストできます

また、Ctrl+ZやCtrl+Shift+Zで、操作を取り消したりやり直したりすることができます

Rec Start

Menu->Rec Startでブラウザ操作を記録することができます

記録できるステップは、URLの移動、クリック、フィールドへの入力です

  • URLの移動
  • クリック
  • ダブルクリック
  • 入力 (input[type=text,textarea])
  • チェック (input[type=checkbox,radio])
  • 選択 (select)

記録後はRec Stopで記録を停止してください

File

組み立てたステップをjson形式で保存することができます

Menu->File->Saveで保存、Menu->File->Openで開きます

Convert

組み立てたステップをテストコードに変換します

実行する言語とブラウザを選択し、テストコードをダウンロードしてください

セキュリティ上の問題でテストコードはtxt形式でダウンロードされます(単体での実行はできますが条件次第で動かないので拡張子を変更してください)

ダウンロードしたテストコードを下記コマンドで実行できます

$ ruby sample.rb

$ python sample.py

Singleton Webdriver

細かいタスクごとにテストコードを作成したい場合この機能が有効です

ダウンロードしたファイルを実行するテストコードと同じ場所に置いてください

通常、Native -> Importステップは個々のテストコード毎に新しいブラウザ(新しいwebdriverのインスタンス)を作成してしまいます

この機能を利用してImportすると、同じブラウザ(同じwebdriverのインスタンス)から個々のテストコードを実行する事ができます

Navigation

URL操作に関するステップを作成できます

Element

要素の操作に関するステップを作成できます

選択後、ブラウザ上で操作したい要素をクリックしてください

クリックで要素を取得できない場合、Ctrl + Shift + Enterで要素を取得できます

Browser

ブラウザ操作に関するステップを作成できます

スクリーンショットの取得やスクロール操作などがあります

Native

条件分岐やループなど高度な操作を行うステップを作成できます

IfやEachなどは子を作成することができます

Verify

検証を行うステップを作成できます

検証結果はコマンドライン上に表示されます

selenium-webdriver

ブラウザ操作をするSeleniumをinstallします

$ gem install selenium-webdriver

chromedriver

SeleniumでChromeを操作する為にchromedriverをinstallします

Macの場合、下記コマンドで

$ brew install chromedriver

もしくはこちらから現時点の最新(v2.9)をダウンロードしてパスを通してください

imagemagick

Firefox以外のブラウザでスクリーンショットを撮る場合、見えてる範囲しか撮れません

画面全体を撮る為にimagemagickをinstallします

Macの場合、下記コマンドで

$ brew install imagemagick

rmagic

Rubyでimagemagicを使用する為rmagickをinstallします

$ gem install rmagick

python

こちらからpython2.x系をダウンロードし、installしてください。

PythonとPython/Scriptsにパスを通してください

下記コマンドでpythonが使えることを確認してください

$ python --version

easy_install

pythonのライブラリをインストールできるツールです

こちらをダウンロードし、pythonで実行してください

$ python ez_setup.py

実行後、easy_installが使えることを確認してください

$ easy_install --version
